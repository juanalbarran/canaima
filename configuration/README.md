# Configurations

Here there are the predefined configurations
A configuration is a set of modules/software that will achieve a working workflow or a specific look and feel.

There are some common workflows between the configurations, but the software to achieve that will change depending of the host or what the configuration want to achieve.

The configurtions will be named after cities i have been.

The principal configuration it will use top confiuration and will not hesistate in resources
