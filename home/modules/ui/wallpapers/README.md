# Wallpaper switcher

This will iterate the wallpapers every 5 mins.
The wallpapers are stored in `~/Pictures/Wallpapers/`
For `1920x1080` or `normal` screens the directory is `~/Pictures/Wallpapers/normal`
For `2560x1080` or `wide` screens the directory is `~/Pictures/Wallpapers/wide`
For `dark` mode the directory is `~/Pictures/Wallpapers/normal/dark`
