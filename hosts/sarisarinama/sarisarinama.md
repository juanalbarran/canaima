# sarisarinama

Older Asus laptop (Laura's). NixOS host.

## Identity

| Field    | Value                         |
| -------- | ----------------------------- |
| Hostname | `sarisarinama`                |
| OS       | NixOS                         |
| Preset   | `caracas`                     |
| Profile  | `playa-caribe` (user: `juan`) |

## Bootloader

GRUB on `/dev/sda` with `useOSProber = true` (detects other OSes on the disk).

## Composition

```
hosts/sarisarinama/default.nix
├── hardware/hardware-configuration.nix   ← generated by nixos-hardware-configuration
└── configuration/caracas.nix             ← OS preset
      └── home-manager.users.juan → configuration/home-configuration/playa-caribe/
```

## Apply

```bash
sudo nixos-rebuild switch --flake .#sarisarinama
sudo nixos-rebuild dry-activate --flake .#sarisarinama   # dry run
```
