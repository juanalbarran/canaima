# canaima

Personal laptop. NixOS host.

## Identity

| Field    | Value                    |
| -------- | ------------------------ |
| Hostname | `canaima`                |
| OS       | NixOS                    |
| Preset   | `budapest`               |
| Profile  | `playa-el-agua` (user: `juan`) |

## Bootloader

systemd-boot with EFI (`canTouchEfiVariables = true`).

## Services

| Service          | Detail                                               |
| ---------------- | ---------------------------------------------------- |
| Display manager  | `ly`                                                 |
| OpenSSH          | enabled                                              |
| Keyboard layout  | `us` (no variant), set via `xserver.xkb`             |
| Lid switch       | `ignore` on all cases (closed lid does not suspend)  |
| `KillUserProcesses` | `false` — user processes survive session logout   |

## System packages

`wget`, `git` — minimal, everything else is managed by Home Manager.

## Composition

```
hosts/canaima/default.nix
├── hardware/hardware-configuration.nix   ← generated by nixos-hardware-configuration
├── services.nix                          ← services listed above
└── configuration/budapest.nix            ← OS preset
      └── home-manager.users.juan → configuration/home-configuration/playa-el-agua/
```

## Apply

```bash
sudo nixos-rebuild switch --flake .#canaima
sudo nixos-rebuild dry-activate --flake .#canaima   # dry run
```
